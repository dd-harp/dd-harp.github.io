A computational challenge for exDE is the computation of terms that vary or parameters that are passed to the derivatives function. The following is our solution to make modular computation:

  • To compute \(x_\ell\) for any term or parameter we need at lag \(\ell\), we create a new variable, called a history variable, \(X_x\)

  • We compute its derivative \[\frac{d X_x}{dt} = x\]

  • We write a function \(F_x,\) which uses the lagderivs function from deSolve:

F_x = function(t, y, pars){
   if (t < 0)
    pars$MYZpar$f0
  else
    lagderivs(t, nr = pars$Xf_ix)
}
  • When we need it, we call \(F_x(t-\ell, y, \mbox{pars})\)

Example

dMYZdt.eg = function(t, y, pars){with(pars,{
  
  kappa = F_kappa(t) 
  M = y[1] 
  Y = y[2] 
  Z = y[3] 
  if (t <= eip) {
    Y_eip <- 0 
    kappa_eip = F_kappa(t)
  } else {
    Y_eip <- lagvalue(t - eip, nr=2)
    kappa_eip = lagderiv(t - eip, nr=4)
  } 
  
  dM = Lambda - g*M 
  dY = f*q*kappa*(M-Y) - g*Y
  dZ = f*q*kappa_eip*(M-Y_eip) - g*Z
  derivs = list(c(dM, dY, dZ, kappa))
  return(derivs) 
})}
F_kappa = function(t){
  .3*(3 + sin(2*pi*t/90))/6 *  exp(.1*(t-200))/(1+exp(.1*(t-200)))
  } 
t=seq(0, 300, length.out=300)
plot(t, F_kappa(t), type= "l")
