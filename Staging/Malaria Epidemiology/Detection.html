  • Blood volume, as measured by the number of red blood cells is a function of age, \(B(a)\). In an adult human, there are about 5 litres of blood, and the normal range is around \(5 \cdot 10^6\) RBCs per microliter (4.69-6.1 in men, 4.04-5.48 in women), so \(B(a) \approx 2.5 \cdot 10^{13}.\)

One microliter of blood is 1/10

P = seq(-5, 5, length.out=100)
detect = function(P, a=2, b=2){
  exp(a*P)/(b+exp(a*P))
}

P2C = function(f, MX=2.5*10^13, s=10^-6){
  10^detect(log10(f*MX*s)) 
}

  • The volume of a blood sample, \(b\) is approximately 1 \(\mu l\) of blood, or \(10^{-6}\), which represents a fraction of \(b/B(a)\) of all blood.

  • The fraction of infected red blood cells (iRBCs), denoted \(f\), is bounded between 0 and 1, so \(P(a) = f(B(a))\)

rPf = function(n, p, MX=11, a=.1, b=.01){
  # MX is the upper bound of realistic values 
  # p is log10 parasite densities
  var = a + b*p
  10^(rbeta1(n, p/MX, var/MX^2)*MX) 
}

rbeta1 = function(n, mu, var){
  rbeta(n, mu^2*(1-mu)/var-1, mu*(1-mu)^2/var-1)
}
