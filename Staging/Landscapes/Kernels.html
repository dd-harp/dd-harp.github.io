kfMake.exp

The function kFmake returns a weight \(w_{i,j}\) for a distance \(d_{i,j}\) from the family of functions:

\[\begin{equation} w_{i,j} = \omega_j \left( e^{-k \left( \frac{d_{i,j}}{s}\right)^\gamma} \right). \end{equation}\]

To code these up, we break the function into two parts. By setting `NORM=FALSE1, it returns a value whos maximum is \(1\), for plotting.

kFmake.exp = function(k=1, s=2, gamma=1,  w=1){
  return(function(dd, w=1){
    wij = w*(exp(-k*(dd/s)^gamma))
    wij/max(wij)
  })
}

kfMake.pwr

\[\begin{equation} w_{i,j} = \frac{\omega_j \epsilon}{\left(d_{i,j} + s\right)^\delta} \end{equation}\]

kFmake.pwr = function(delta=1, s=1, w=1){
  return(function(dd, w=1){
    wij = w/(dd+s)^delta
    wij/max(wij)
  })
}

kFmake.mix

This takes a sum of two of the previous functions:

kFmake.mix = function(p=0.9, k=1, s=2, gamma=1, delta=1, w=1){
  return(function(dd, w=1){
    wij = p*w*(exp(-k*(dd/s)^gamma)) + (1-p)*w/(dd+s)^delta
    wij/max(wij)
  })
}
par(mfrow = c(1,2))
dd = seq(0,5, length.out=200) 
k1 = kFmake.mix(.9, 2, 2, 1, .5)
k2 = kFmake.mix(.5, 2, 2, 1, .5)
k1(dd) -> w1 
k2(dd) -> w2 
plot(dd, w1, type = "l", ylab = "Weights, Scaled", xlab = "Distance")
lines(dd, w2)

k1 = kFmake.mix(.9, 2, 2, 1, .5)
k2 = kFmake.mix(.5, 2, 2, 1, .5)
plot(dd, w1/sum(w1), type = "l", ylab = "Weights, Normalized", xlab = "Distance")
lines(dd, w2/sum(w2))
